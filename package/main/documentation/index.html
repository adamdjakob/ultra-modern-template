Getting Started

Quick Start Guide
1. File Setup

Extract the downloaded zip file to your desired location.

2. Dependencies

The template includes all necessary dependencies:

  • Bootstrap 5.3.0
  • Font Awesome 6.0.0
  • AOS (Animate On Scroll)
  • Swiper.js
  • Particle.js
3. Running the Template

Open index.html in your web browser to view the template.

Customization

Colors

The template uses CSS variables for easy color customization:


:root {
    --primary-color: #6b46c1;
    --secondary-color: #718096;
    --dark-color: #2d3748;
    --light-color: #f7fafc;
}
Typography

The template uses Google Fonts:

  • Primary Font: 'Inter', sans-serif
  • Heading Font: 'Poppins', sans-serif
