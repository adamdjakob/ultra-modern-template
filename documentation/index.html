Ultra Modern Documentation

A comprehensive guide to using and customizing the Ultra Modern HTML template.

⚡ Quick Start (5 Minutes)

  1. Download & Extract

    Extract the template files to your project directory

  2. Open index.html

    Double click index.html or serve it through a local server

    python -m http.server 8000

    Edit these files for basic changes:

    • css/style.css - Colors and styles

    • js/main.js - Interactions and effects

Getting Started

Quick Start Guide
1. File Setup

Extract the downloaded zip file to your desired location.

2. Dependencies

The template includes all necessary dependencies:

  • Bootstrap 5.3.0
  • Font Awesome 6.0.0
  • AOS (Animate On Scroll)
  • Swiper.js
  • Particle.js
3. Running the Template

Open index.html in your web browser to view the template.

Customization

Common Customization Examples
1. Changing the Hero Background Video


<div class="hero-video-wrapper">
  <video autoplay muted loop id="heroVideo">
    <source src="your-video.mp4" type="video/mp4">
  </video>
</div>
2. Modifying the Particle Effect

// In js/main.js
particlesJS('particles-js', {
  particles: {
    number: {
      value: 80, // Change number of particles
      density: {
        enable: true,
        value_area: 800
      }
    },
    color: {
      value: '#ffffff' // Change particle color
    }
  }
});
3. Adding Custom Animations

/* In css/style.css */
.custom-animation {
  animation: float 6s ease-in-out infinite;
}

@keyframes float {
  0% { transform: translateY(0px); }
  50% { transform: translateY(-20px); }
  100% { transform: translateY(0px); }
}

Troubleshooting FAQ

Common causes and solutions:

  • Check if video format is supported (use MP4)
  • Ensure autoplay, muted, and loop attributes are present
  • Try converting video to a different codec (H.264)

<video autoplay muted loop playsinline>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
</video>

Common fixes:

  • Ensure viewport meta tag is present
  • Check Bootstrap breakpoint classes
  • Verify media queries

<meta name="viewport" content="width=device-width, initial-scale=1.0">

/* Responsive fixes */
@media (max-width: 768px) {
  .hero-section {
    height: auto;
    min-height: 100vh;
  }
}

Troubleshooting steps:

  • Check if AOS is properly initialized
  • Verify data-aos attributes
  • Check for JavaScript console errors

// Initialize AOS
document.addEventListener('DOMContentLoaded', function() {
  AOS.init({
    duration: 1000,
    once: true,
    offset: 100
  });
});
