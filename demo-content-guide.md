# Demo Content Creation Guide

## Required Preview Images

### Main Preview Image (590x300)
1. Full homepage screenshot
2. Show key features
3. Modern and clean design
4. High quality, optimized

### Feature Screenshots
1. Hero Section
   - Video background
   - Particle effects
   - Call to action

2. Portfolio
   - Grid layout
   - Filtering
   - Project modal

3. Services
   - 3D flip cards
   - Icon animations
   - Hover effects

4. Team
   - Member cards
   - Social links
   - Hover effects

5. Blog
   - Card layout
   - Featured images
   - Category tags

6. Contact
   - Multi-step form
   - Map integration
   - Contact info

### Responsive Previews
1. Desktop (1920x1080)
2. Tablet (768x1024)
3. Mobile (375x812)

## Image Sources
1. Unsplash.com
2. Pexels.com
3. Custom screenshots
4. Portfolio mockups

## Image Optimization
1. Compress all images
2. Use WebP format
3. Maintain quality
4. Responsive sizes

## Organization
1. Separate folders for:
   - Preview images
   - Portfolio images
   - Blog images
   - Team photos
   - Background images

2. File naming convention:
   - preview-main.jpg
   - feature-hero.jpg
   - portfolio-1.jpg
   - blog-1.jpg
   - team-1.jpg
